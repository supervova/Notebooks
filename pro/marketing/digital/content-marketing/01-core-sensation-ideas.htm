Идеи «гвоздей» для корпоративного блога и сайта в целом

См. также идеи для корпоративного журнала, советы по контент-маркетингу и генератор идей содержания для блога / СМИ

Любой сайт (в том числе — коркорпоративный) должен в первую очередь быть интересным для ЦА. То есть не размениваться на победные реляции, а содержать полезные и развлекательные материалы для клиентов или профессионального сообщества (с точки зрения SEO — linkbit-ы, приманки для ссылок). Формат материалов может быть разным: обычная заметка, «шпаргалка», инфографика (примеры; формы визуализации), интерактивная инфографика, аудиоподкаст, видеоурок, игра, электронная книга, шаблоны документов и презентаций, онлайн-служба, шутки в различном исполнении — простой текст, фотография с комментарием (демотиваторы и пр.), флеш-ролик. Как правило, под такие материалы отводится специальный раздел («Проф. сообщество», «Другие интересы», «Жизнь вокруг нас» и т.п.) и, в частности, корпоративный блог.

Процесс создания «гвоздей» прост.

  1. Поиск тем.
    • По Zite, Flipboard, StumbleUpon и генератору идей и т. д. Особенно интересны — набирающие популярность, но пока не вышедшие в тираж.
    • Метод случайных связей. Ключевая фраза + случайно выбранная из несмежной области тема. Например, форекс + иллюзионист; форекс + соц-медиа; форекс + кулинария.
    • Мозговой штурм — адаптация под тематику своего сайта или сайта клиента.
  2. Определение формата:
    • по матрице воронки продаж.
    • Bookmarks > Webmastering > Inspiration > Features

Некоторые идеи


  • Сервис по автоматизации какой-то обычной для ЦА процедуры. Например, сервис рассылки пресс-релизов для PR-компаний. Это может быть онлайн-приложение. Или панели дополнительных инструментов Программные модули, расширяющие возможности браузеров, устанавливаемые пользователями по желанию. для браузеров Chrome, Firefox, Internet Explorer, Yahoo, мини-приложения Facebook. Для начала нужно выяснить чего ЦА не хватает в браузерах. Например для пиарщиков, навскидку, это могли бы быть индикаторы упоминаемости в интернете слов и выражений, заданных пользователем (отдельно — данные по веб-ресурсам традиционных СМИ). Например, пиар-консультант или его клиент хочет проследить, как в ходе кампании меняется количество упоминаний «Елены Берковой», причем в сравнении с «Дженой Джеймсон». Пожалуйста: каждый день в его браузере под адресной строкой меняются показатели двух разноцветных шкал — наводишь на них курсор, получаешь абсолютные цифры.
  • Профессиональные рейтинги и награды. Пример.
  • Один из самых эффективных жанров в контент-маркетинге — «Введение в [тему]» или «[Тема] для начинающих]». Пример.
  • Списки типа «10 простых шагов», «10 мифов», «10 вещей, которые я ненавижу» etc — всё по теме сайта. Например, «10 самых известных кампаний черного пиара» или «10 несостоявшихся наследников Ельцина и Путина» etc.
  • Списки или подборки картинок типа «Короткая история веб-дизайна», «Короткая история рекламы», «Короткая история туалетов». См. пример. Еще лучше собирать такие материалы в виде досок на Пинтерест, а в корпоративном блоге делать анонс.
  • Справочники и инструкции по тематике отрасли: «Как сделать то-то и сё-то». Например, «Как организовать презентацию», «Как написать 10 страниц пресс-релиза за 1 минуту» etc.
  • Визуализация данных: инфографика (примеры; формы визуализации), инфовидео (пример), инфострукции (см. пердыдущий пункт).
  • Новый отраслевой термин и статьи о нем (в том числе — в Википедии). Пример.
  • Интервью жанра «мастер-класс» с ключевыми фигурами отрасли, лидерами конкурирующих организаций (обязательно с фото).
  • Опросы мнения посетителей сайта по текущим проблемам современности: «Вам кто больше нравится: Дмитрий Медведев или весна?», «Кто оправдал свое прозвище в минувшем футбольном сезоне: „мясо“, „кони“ или „бомжи“?» etc.
  • Постоянные тесты по теме. Например, «Ваш потенциал: рядовой избиратель, пиар-советник, государственный деятель», «Сколько раз вами манипулировали пиарщики?» etc.
  • Вакансии.
  • Формы гадания по сайту — по аналогии гадания по книге. Человек, вводит свой вопрос, ему выдается случайно выбранная строка из текстов сайта, а также — случайная строка из Википедии и снимок с Flickr.
  • UGC-конкурсы (user generated content): фото и видео. Пример — конкурс самодельных рекламных роликов чипсов Doritos.
