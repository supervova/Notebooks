Добавление/редактирование/удаление содержания

  1. Меню быстрых команд > + Новая страница.
  2. Выбираем тип материала.
  3. Если страница структурная, для нее нужно выбрать пункт в меню.
  4. Чтобы исправить тип материала
    > Structure > Content types > в строке нужного типа: edit | manage fields | manage display | delete
  5. Чтобы добавить тип материала
    > Structure > Content types > Add Content type
  6. Чтобы материал немедленно появился на сайте
    > Configuration > Perfomance
    Нажать Clear all caches
