template.php

Предназначен для любой условной логики, функций обработки данных (в том числе для изменения генерируемого HTML) — чтобы шаблоны tpl.php содержали только HTML и php-переменные. Хорошая практика — создавать здесь функции, которые будут передавать переменные щаблонам. Должен начинаться с <?php, а закрывающий тег рекомендуется опускать.
