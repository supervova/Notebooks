Блоки и регионы

> Structure > Blocks

  • Регионы — это области на странице сайта, заданные автором темы и обозначенные в разметке, как переменные ($sidebar_right;). Блоки — это содержание регионов: фрагменты информации (как системной, так и редакторской).
  • Для того, чтобы менять макет, можно использовать условную конструкцию. Наример, чтобы показывать боковую панель только на внутренних страницах, пишем в шаблоне:
    <?php
  if($page['sidebar']) {
    // do something
  }
?>
    Чтобы проверять есть ли содержание в регионе, описанном в шаблонах node, вида или блока, надо в template.php прописать специальную функцию.
  • Пример программной подготовки содержания для региона (краткое содержание правой колонки)
  • Для сайтов заказчиков делать блок «Справка» и показывать его только по адресам node/*/edit and node/add/* (Но, в принципе, может быть, будет достаточно оставить административной темой Batrik, в которой включенный по умолчанию модуль Help показывает подсказки).
  • Обязательно составлять для себя блок-схему в OmniGraffle — со всеми блоками, регионами, видами и отношениями между ними.
