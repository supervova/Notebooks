Меню

> Structure > Menus

  • Меню в D — иерархические списки ссылок. По умолчанию, D предлагает четыре меню: Main menu , Management, Navigation и User Menu.
  • C Main menu всё понятно. Navigation, по умолчанию, содержит ссылки для авторизовавшихся пользователей. User Menu — ссылку на профайл и ссылку выхода из системы. Management — ссылки на инструменты редактора.
  • Можно создавать свои меню (например, отдельные меню для каждого раздела).
  • Каждое меню и каждая ссылка могут быть выключены или появляться в зависимости от авторизации или роли пользователя.
