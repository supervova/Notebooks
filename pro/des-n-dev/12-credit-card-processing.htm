Процессинг банковских карт

Диапазон решений по приему онлайн-платежей распространяется от простого делегирования обязаностей PayPal или Google Checkout до обработки шифрованных данных собственными силами.

  1. Для новых предприятий больше подходят Payment-Processing сервисы — например, PayPal или Google Checkout. Хотя их услуги дороже, а процесс рассчетов не самый простой, подключиться к таким системам гораздо проще.
  2. Другое принципиальное решение — IMA-PSP:
    1. Надо в своем банке получить Internet Merchant Account (IMA) ID.
    2. Перед тем, как подлписать контракт на обслуживание, нужно внимательно изучить тарифные планы.
    3. Не рекомендуется обрабатывать платежи с карт вручную.
    4. Затем нужно выбрать PSP (payment service provider; другое название — payment gateway), изучить их тарифы и решить, что будет выгоднее и удобнее для потребителей — расплачиваться на сайте магазина или на сайте PSP. Неокторые PSP предлагают IMA ID, а некоторые банки — PSP. Проверочный лист PSP:
      • SSL-шифрование всех страниц платежного процесса
      • Процедуры авторизации
      • API для передачи данных с сайта магазина в шифрованном виде
      • API для определения подтверждения/отказа
      • Автозаполнение уже заполненных покупателем полей
      • Возможность фирменного дизайна страниц платежных форм.
      • Переадресация на сайт магазина по окончании рассчета.
      Некоторые сервисы, отвечающие этим требованиям:
      • Authorize.Net
      • 2Checkout
      Краткий перечень преимуществ этих систем.

Процесс

  1.  SSL-защищенная корзина
  2. Последовательность форм рассчета
  3. Введенные данные по шифрованному каналу передаются в базу, затем, опять в шифрованном виде, — на PSP-сайт. Пользователь также перенаправляется на сайт PSP.
  4. Пользователь заполняет еще незаполненные поля на PSP-сайте. PSP-сайт отправляет статус транзакции, ID заказа и др. платежную информацию назад, на сайт магазина. Все операции должны соответствовать стандартам безопасности PCI DSS.
  5. Пользователь возвращается на страницу подтверждения/отказа на сайте магазина.
