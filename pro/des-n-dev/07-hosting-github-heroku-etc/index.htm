Хостинг: github, Heroku, AWS и др.

Веб-сервис для хостинга проектов github и облачные хостинги стали привлекательной альтернативой традиционному виртуальному и выделенному виртуальному хостингу. На github можно бесплатно хостить небольшие статичные сайты. «Облака» предоставляют лучшее соотношение цена/качество для растущих и потенциально крупных проектов.

Cистема контроля версий Git и веб-сервис для хостинга проектов github

github — веб-сервис для хостинга проектов и их совместной разработки, социальная сеть для разработчиков со страницами, с небольшими вики и системами отслеживания ошибок для каждого проекта. Кроме всего прочего прочего предлагает хостинг статичных веб-страниц github Pages: до 300Мб дискового пространства для статики и всех версий — бесплатно. Для публикации используются веб-интерфейс или ПК-приложения для Mac и Windows. Подходит для небольших и средних веб-сайтов, при необходимости сайт легко переносится на облачный хостинг.

Возможности сайтов github Pages можно расширить за счет:

  • генератора статисных сайтов Jekyll (можно попробовать также специальный веб-редактор для Jekyll — Prose); кроме того статических страниц, Jekyll создает RSS-ленту.
  • системой комментариев Disqus
  • социальными плагинами Facebook
  • фотогалереями фотохостинга Imgur
  • сниппетом для тегов
  • пользовательским поиском Google.

Для подключения доменных имен в корне репозитория создать файл CNAME, а в нем написать домен. Если это под-домен, то нужно прописать CNAME-запись, которая будет указывать на username.github.com. А если это домен верхнего уровня — A-запись, указывающую на 207.97.227.245 (CNAME для доменов верхнего уровня лучше не использовать...). Подождать, пока обновятся DNS.

Облачный хостинг

Популярные сервисы и их главные преимущества:

  • Heroku — простота и свобода; небольшой сайт/приложение можно хостить бесплатно.
  • Amazon Web Services (AWS) — наибольшая свобода для разработчиков.
  • Linode — в некоторых случаях дешевле AWS.
  • AppFog (PHPFog) — в некоторых случаях дешевле Heroku.
  • Google Code Engine — интеграция с сервисами Google.

Сравнение цен. Приложение/сайт с БД на 10 МБ и одним рабочим процессом: Heroku — $75 в месяц; AWS — от $17,69 в месяц  (предоплата за 3 года) до $57,6 в месяц.

Heroku и Google Code Engine относятся к PaaS модели. Amazon Web Services — к IaasS.
