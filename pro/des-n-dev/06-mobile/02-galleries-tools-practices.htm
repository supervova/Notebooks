Не просто мобильный веб-сайт, а веб-приложение

У веб-приложений есть свои преимущества: совместимость с разными мобильными платформами; обычный html-css-javascript, а не Objective-C или Java; не надо делить с Apple или с кем-то ещё прибыль; свобода от жестких рамок требований Apple и др. Есть и недостатки: у веб-приложений пока нет единого рынка вроде iTunes Store; нельзя пользоваться многими возможностями мобильных операционных систем — прежде всего, связанных с интерфейсом (доступ к камере, акселерометр). Впрочем, почти все недостатки решаются с помощью онлайн-сервиса PhoneGap, с помощью которого можно конвертировать веб-приложения в «родные» приложения Apple, Android, BlackBerry, Simbian (Windows ожидается).

Tools to design a website for iPhone

  • iPhoney is a tool to see how your web creations will look on iPhone.
  • Test iPhone A web browser based simulator.
  • iWebKit is a file package designed to help you create your own iPhone and iPod Touch compatible website or webapp.
  • iUI is a framework consisting of a JavaScript library, CSS, and images for developing iPhone web designs.
  • Intersquash. You can instantly create an iPhone compatible site from your RSS feed url.
  • WPtouch transforms your WordPress blog into an iPhone application-style theme, complete with ajax loading articles and effects, when viewed from an iPhone or iPod touch.
  • Jaipho Gallery If your website features a photo gallery, Jaipho will optimize it for iPhone users.
