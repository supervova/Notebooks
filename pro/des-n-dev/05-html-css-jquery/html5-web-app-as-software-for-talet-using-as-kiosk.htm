Терминал самообслуживания: планшет + веб-приложение на HTML5

Чтобы превратить планшет в терминал самообслуживания, следует сконструировать антиворовское крепление и разработать веб-приложение.

  • Спрятать GUI браузера с помощью <meta name="apple-mobile-web-app-capable" content="yes" />
  • Вставить ссылку на иконку для рабочего стола <link rel="apple-touch-icon" href="/path/to/icon.png" />
  • Установить цвет статус-бара (по умолчанию — черный) <meta name="apple-mobile-web-app-status-bar-style" content="black" />
  • Заменить обработчики событий с hover на touch.

См. также общие принципы разработки мобильного приложения, helper.js и extended.htm.
