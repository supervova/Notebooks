HTML5 — Атрибуты

data-

Наиболее значимая новинка HTML5. Позволяет метить элементы по их назначению, не прибегая к оформительскому атрибуту class, а также назначать некоторые свойства, которые впоследствии можно использовать в JS. Название атрибута состоит из префикса и пользовательского значения. Например, data-name, data-gender, data-personality, data-appearance и т.д.
<div data-type="sprite" data-offsetY="100" data-Xposition="50%" data-speed="2"></div>
