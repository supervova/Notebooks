Zen coding

Вводим сокращение, нажимаем ctrl + , (cmd + ,) или Tab (в Sublime Text, в DW для Windows) — получаем код. (В Mac OS, в DW > Keyboard Shortcuts надо переопределить комбинацию cmd + , по умолчанию присвоенную команде Edit > Go to Line).

  • E — элемент: article, p.
  • E#id — элемент с id: section#content, p#intro, span#error. Если элемент не указан — будет выводиться DIV.
  • E.class — элемент с классом: div.header, p.error.critial. Можно создавать элементы с классом (классами) и id: #content.column.width (также, как и в случае с id, если элемент не указан, будет выводиться DIV).
  • E[name=value] — элемент с другими атрибутами: a[href=http://sms-roman.ru], a.btn[href=#].
  • E{text} — элемент с каким-то содержанием: p.alert{Lorem ipsum}.
  • E>E — вложенный элемент: div>p, #footer>p>span.
  • E+E— последовательные элементы: h1+p, header#header+#content+footer#footer.
  • E*N — множитель: p*4, ul#nav>li*3>a.
  • (E>E)+E — обособление вложенных элементов от последовательных: .doc-container>(header.doc-header+.doc-body+footer.doc-footer)
  • E$*N — добавление порядкового номера: ul#nav>li.menu-0$*3 = <ul id="nav"><li class="menu-01"></li><li class="menu-02"></li><li class="menu-03"></li></ul>.

Emmet

Расширяет возможности «стенографии» в HTML и добавляет возможность «стенографии» в CSS/LESS.

  • p20 = padding: 20px;
  • ml20 = margin-left: 20px;
  • m10-auto = margin: 10px auto;
  • w50 = width: 50px;
  • w50p = width: 50%;
  • lorem20 = 20 слов из Lorem Ipsum
  • link:css = <link rel="stylesheet" href="style.css" media="all">
  • link:favicon = <link rel="shortcut icon" type="image/x-icon" href="favicon.ico">
  • link:rss = <link rel="alternate" type="application/rss+xml" title="RSS" href="rss.xml">
  • meta:utf = <meta http-equiv="Content-Type" content="text/html;charset=UTF-8">
  • input:text = <input type="text" name="" id="">
  • html:5 =
