CSS3. 3D

Реализуется с помощью свойства transform, которое используется также для двухмерного перемещения, вращения, масштабирования, перекашивания.

Значения:

  • matrix3d
  • perspective
  • rotateX, rotateY, rotateZ, rotate3d
  • scaleX, scaleY, scaleZ, scale3d
  • translateX, translateY, translateZ, translate3d

Вращение

Пример

.rotateZ img {

-moz-transform: rotateZ(45deg);

-ms-transform: rotateZ(45deg);

-o-transform: rotateZ(45deg);

-webkit-transform: rotateZ(45deg);

}

Перемещение

Кроме перемещения по осям x и y, можно перемещать объект по оси z, а также использовать сокращенную запись translate3d(4px, 4px, 4px): первая цифра — по оси x, вторая — y, третья — z.

Пример

.translateZ img {

-moz-transform: translateZ(-20px);

-ms-transform: translateZ(-20px);

-o-transform: translateZ(-20px);

-webkit-transform: translateZ(-20px);

}

Масштабирование

Кроме масштабирования по осям x и y, можно увеличивать/уменьшать объект по оси z, а также использовать сокращенную запись scale3d(.5, 1.5, .75). Коэффициенты умножаются на линейные размеры. В случае с масштабированием по оси z — сразу на ширину и высоту.

Перспектива

Чем меньше коэффициент, тем сильнее искажение. Пример

.perspective25 img {

-moz-transform: perspective(25);

-ms-transform: perspective(25);

-o-transform: perspective(25);

-webkit-transform: perspective(25);

}

.perspective50 img {

-moz-transform: perspective(50);

-ms-transform: perspective(50);

-o-transform: perspective(50);

-webkit-transform: perspective(50);

}

С помощью свойства perspective-origin можно менять точку зрения, которая по умолчанию располагается в чентре, спереди.

3D для потомков

Чтобы потомки элемента с эффектом 3D трансормировались соответственно, надо им присвоить свойство transform-style: preserve-3d.

Изменение опорной точки

transform-origin: right top;
