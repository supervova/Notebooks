HTML5

  1. Элемент <a> стал блок блоковым. То есть теперь в него можно заключать «шапки», не графические баннеры и т.д.
  2. Благодаря новым атрибутам в будущем можно отказаться от JavaScript и плаг-инов (а теперь использовать Modernizr для реализации этих атрибутов во всех браузерах)
    • Типы полей input email, url, date и datetime имеют встроенную проверку формата ввода. Урок по форме HTML5
    • Атрибуты autofocus и placeholder (текст в поле, исчезающий в фокусе — например, label или подсказка)
    • Элементы canvas (векторная графика — например, графики — с помощью JavaScript-а), audio и video
  3. Анимировать графику в canvas можно с помощью Cake — js-библиотеки Google.

Видео 

<video controls="true" height="295" width="480">
<!-- hello iOS, Safari and IE9 -->
<source src="chris.mp4" type="video/mp4">
<!-- Hello Chrome and Firefox (and Opera?) -->
<source src="chris.webm" type="video/webm">
<!-- Hello Firefox and Opera -->
<source src="chris.ogv" type="video/ogg">
<!-- Hello legacy -->
  Your browser does not support the video tag, <a href="http://www.youtube.com/watch?v=IhkUe_KryGY">check the video on YouTube</a>.
</video>

Атрибуты элемента video:

  • src
  • controls
  • poster — значением будет путь к картинке-обложке
  • autoplay
  • width
  • height

Vid.ly — бесплатный сервис конвертации и хранения. Инвайт-Код HNY2011.
В результате конвертации не нужно писать столько кода, как при самостоятельном аплоде-хостинге, только
<video src="http://vid.ly/4f3q1f?content=video" controls></video>
