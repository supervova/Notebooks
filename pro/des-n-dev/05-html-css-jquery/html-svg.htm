SVG

SVG — формат векторной графики, использующий XML для описания изображений. В CSS преимущественно используется в качестве спрайтов.

  1. Файл создается в Illustrator-е и сохраняется, как SVG. Наиболее универсальные параметры:
    • SVG Profiles: SVG 1.1
    • Fonts
      • Type: SVG
      • Subsetting: Only Glyphs Used
    • Images
      • Location: Embed
    • Флаг Preserve Illustrator Editing Capabilities не ставится.
  2. Затем надо открыть файл в Dreamweaver-е и исправить единицы измерения в размерах в теге SVG: вместо px — em (причем, сохраняя сами значения). 
    <svg version="1.1" id="Layer_1" xmlns="http://www.w3.org/2000/svg"
xmlns:xlink="http://www.w3.org/1999/xlink" x="0px" y="0px"
width="30px" height="160px" width="30em" height="160em"
viewBox="0 0 30 160" enable-background="new 0 0 30 160" xml:space="preserve">

Браузеры:

  • Internet Explorer 9+
  • Firefox 4+
  • Chrome 4+
  • Safari 4+
  • Opera 9.5+

Для поддержки в IE6-8 можно использовать хаки и PNG.
