Модуль imgopt: оптимизация картинок без потери качества

Оптимизация

  1. Manage Sites > [ сайт ] > [ редактировать ] > Advanced Settings > Design Notes > Clean Up Design Notes
  2. Переход к папке с файлами. Например,
    cd /Users/vova/Sites/polylog-old/a/img
    Оптимизировать все файлы в выбранной директории и поддиректориях
    imgopt .
    Оптимизировать только файлы одного типа только в выбранной директории
    imgopt *.png
    Оптимизировать один файл
    imgopt girl.png
    Оптимизация может занять несколько часов

Установка

  1. Finder: переход к папке /usr/local/bin
  2. Переписать туда imgopt
  3. Установить нужные права (здесь и далее — команды в Терминале)
    chmod a+x /usr/local/bin/imgopt
  4. Компилировать программу jfifremove и переместить ее в ту же папку
    cd [путь к папке, куда был скачен jfifremove]
    gcc -o jfifremove jfifremove.c
    sudo mv jfifremove /usr/local/bin/
  5. Finder: переход к папке /usr/local/lib. Если там нет файлов libjpeg*, установить эту библиотеку (см. soft/web-development/optimg/
  6. Проверить в терминале, установлена ли коллекция утилит AdvanceCOMP:
    advdef
    Если нет, скачать коллекцию, разархивиаровать и установить
    cd [путь к папке с разархивированной коллекцией/]
    sudo ./configure
    sudo make install
  7. Проверить в терминале, установлена ли утилита OptiPNG:
    optipng
    Если нет, скачать, разархивиаровать и установить
    cd [путь к папке с разархивированной утилитой/]
    sudo ./configure
    sudo make install
  8. Проверить в терминале, установлена ли утилита PNGout:
    pngout
    Если нет, скачать, разархивиаровать, скопировать исполняемый файл pngout
    Finder: переход к папке /usr/local/bin
    Вставить pngout
